注意

  • synchronized总是与对象关联
  • 如果方法是静态方法，则关联对象是class对象
  • 如果方法是非静态方法，则关联对象是当前实例
  • 允许使用synchronized修饰abstract方法是没有意义的，因为synchronized是属于实现的一部分

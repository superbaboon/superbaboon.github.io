如何编写和发布博客

Mar 7th, 2016

开启本地监听和http服务器

  • rake watch — 监听源文件变化
  • rake preview — 本地开启http端口，通过访问http://localhost:4000/%E8%AE%BF%E9%97%AE

新建文章

  • rake new_post[‘{post name}’]

注意：文件名命名规则，参考这里

发布

  • rake generate — 根据源文件生成目标文件
  • rake deploy — 发布到github
  • git push origin source — 记录修改内容

Done！
